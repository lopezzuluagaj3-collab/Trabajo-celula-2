# Trabajo-celula-2
MiniOS

MiniOS es un proyecto donde creamos un “mini sistema operativo” usando solo HTML y CSS.
No usamos JavaScript, así que todo funciona de manera muy simple.

En este proyecto tienes una página principal (index.html) que muestra diferentes “aplicaciones” dentro de un iframe (una ventanita donde se carga otra página).
El usuario puede cambiar de aplicación usando un menú hecho con HTML.

¿Qué puedo hacer en MiniOS?

Ver una página principal que parece un sistema operativo.
Cambiar entre diferentes aplicaciones (como Notas, Tareas o Clima).
Todo se ve dentro de un iframe.
Navegar sin nada de JavaScript, solo con HTML.

¿Qué es cada cosa?

index.html: aquí ves la interfaz principal del MiniOS.
apps: contiene las diferentes aplicaciones, cada una es un archivo HTML.
styles: aquí está el archivo CSS para darle estilo al MiniOS.
